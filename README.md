## 🧪 Local Development

### 1. Clone the repository

```bash
git clone https://github.com/minamrahman727/smrehmana.git
cd smrehmana
```
### 2. Install dependencies

```bash
npm install
# or
yarn
```
### 3. Start the local server
```
bash
npm run dev
# or
yarn dev
```  
- Navigate to http://localhost:3000 to view the site.
